# 📘 Teoría de Conjuntos: Algorithms & Logic
*Sintetización Programática de Lipschutz, Halmos y Jech*

---

### 🚀 Descripción del Proyecto
Este repositorio es un laboratorio de *ingeniería inversa* aplicado a la Teoría de Conjuntos. Transforma fundamentos matemáticos abstractos en algoritmos ejecutables de Python, optimizando la comprensión de colecciones mediante programación funcional y entornos Jupyter. El proyecto documenta la transición desde la lógica formal de los axiomas hacia estructuras de datos eficientes y escalables para el análisis computacional moderno.

### 🧠 Método de Aprendizaje
Aplicamos un enfoque de *aprendizaje activo* y *transcripción consciente*:
1. *Análisis Teórico:* Desglose capítulo a capítulo de las tres "biblias" de la disciplina.
2. *Codificación Manual:* Transcripción directa en *VS Code* para fortalecer la memoria sintáctica.
3. *Documentación Interactiva:* Uso de Notebooks para convertir la matemática pura en lógica viva.



### 📂 Estructura del Repositorio
Para mantener el estándar de *Clean Architecture*, el proyecto se organiza así:

* src/: Scripts de Python puro con las funciones core de lógica.
* notebook/: Laboratorios interactivos (.ipynb) con explicaciones paso a paso.
* data/: Conjuntos de datos de prueba para validación de algoritmos.
* Docs/: Notas de estudio y resúmenes de los capítulos.
* ejercicios/: Retos de razonamiento abstracto resueltos con código.
* requirements.txt: Dependencias necesarias para el entorno virtual (venv).
